# 2024 SSG 랜더스 타선 효율 분석
> 타율을 넘어서: OPS·wOBA 기반 득점 기여도 분석

---

## 프로젝트 개요

KBO에서 타자를 평가할 때 가장 많이 쓰이는 지표는 **타율(AVG)**입니다.  
하지만 타율은 볼넷과 장타를 전혀 반영하지 못합니다.

이 프로젝트는 2024 시즌 SSG 랜더스 타자 기록을 수집하고,  
**OPS**와 **wOBA** 지표를 계산해 타율 순위와 실제 득점 기여도 순위의 차이를 분석합니다.

---

## 분석 질문

- 타율이 높은 선수가 실제로 득점에 더 많이 기여했는가?
- OPS 기준으로 과소평가 / 과대평가된 SSG 타자는 누구인가?
- 타선 구성 측면에서 출루형 vs 장타형 분포는 어떠한가?

---

## 사용 데이터

- **출처**: [KBO 공식 기록실](https://www.koreabaseball.com/Record/Player/HitterBasic/Basic1.aspx)
- **대상**: 2024 정규시즌, SSG 랜더스, 200타석 이상 타자
- **수집 방법**: Playwright 비동기 크롤링

---

## 분석 지표

| 지표 | 설명 | 타율 대비 개선점 |
|------|------|-----------------|
| **OPS** | 출루율 + 장타율 | 볼넷, 장타력 반영 |
| **wOBA** | 가중 출루율 | 타격 결과별 득점 기여도 가중치 적용 |
| **ISO** | 순수 장타력 (SLG - AVG) | 장타 생산 능력 분리 |

> wOBA 가중치: 단타 0.89 / 2루타 1.27 / 3루타 1.62 / 홈런 2.10  
> (KBO 2024 시즌 평균 득점 환경 기반 근사치)

## 주요 인사이트

분석 결과, **타율 순위와 OPS 순위가 뒤바뀌는 선수들**이 존재했습니다.  
이는 타율만으로 타자를 평가할 경우 볼넷 능력이나 장타력이 뛰어난 선수를  
과소평가하게 됨을 보여줍니다.

특히 타율이 중하위권이어도 wOBA가 높은 선수의 경우,  
타순 배치나 기용 방식을 재검토할 여지가 있습니다.
